# [SkyBeje](https://skybeje.net/) #
* Skybejeは、WebRTC[SkyWay](https://nttcom.github.io/skyway/)とブラウザ内データベース(IndexedDB)を利用し、新しい形のサービスを模索するための試験的なサービスです。
