配信設定

高解像度／高フレームレートでの利用は、端末に高負荷が掛かるので注意してください。

画面の動きが少ない資料等を表示したい場合は「選択画面の解像度」を選択し、フレームレートは低めで指定してください。

ゲーム等の動きが多い画面をスクリーンシェアしたい場合は、仮想ウェブカメラを使用してライブキャストする事をお勧めします。
仮想ウェブカメラとは、デスクトップの画面をWebカメラとして扱えるソフトウェアです。
ManyCam / ニコ生デスクトップキャプチャー / OBS-VirtualCam といったソフトウェアがあります。
スクリーンシェア機能を利用するには、
以下の Chrome Extension の追加が必要です。

SyBeje Screen Share Extension
